# VCS入門

2013/4/15 高野 将

## はじめに

本文書はVCS（Version Control System：バージョン管理システム）の概念、使い方を、未経験者でもわかるように伝えることを目的としています。本書のゴールは以下の通りです。

- VCSの概念を理解する。
- SCMの考えを理解する。
- SCMを意識したVCSの利用ができる。

なお、なるべく特定のVCSに依存しない説明を行い、ツールの詳細な使い方については本書の対象外とします。別稿として、各VCS毎の使い方はまとめるかもしれません。
